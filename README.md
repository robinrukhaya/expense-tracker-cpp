# Expense Tracker (C++)

A **console-based Expense Tracker** built in **C++** using **Object-Oriented Programming (OOP)** principles.  
This project allows users to **track, categorize, and store their daily expenses** with persistent file storage.

---

## Features

- Add a new expense with **amount, category, and date**
- View all expenses in a structured format
- Persistent storage using **text files** (`data.txt` / `transactions.csv`)
- Multiple project versions:
  - `personal_expense.cpp` (basic version)
  - `with_oop.cpp` (OOP implementation)
- Optional: Executable files included for quick testing

---

## Tech Stack

- **Language:** C++
- **Concepts Used:**
  - Classes & Objects (OOP)
  - File Handling (read/write transactions)
  - Vectors & Arrays
  - Modular programming
